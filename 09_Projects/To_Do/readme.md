# CLI To-Do Application

A command-line based To-Do application built using Python.
This project allows users to manage daily tasks efficiently through a simple and interactive terminal interface.

---

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Persistent storage using JSON
* Menu-driven interface

---

## Tech Stack

* Python
* JSON (for data storage)

---

## Project Structure

```bash
todo_cli/
│
├── todo.py
├── tasks.json
├── utils.py (optional)
└── README.md
```

---

## How to Run

### 1. Navigate to project folder

```bash
cd todo_cli
```

### 2. Run the application

```bash
python todo.py
```

---

## Usage

* Select options from the menu displayed in the terminal
* Enter task details when prompted
* Tasks are automatically saved in `tasks
