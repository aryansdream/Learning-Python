# Python Learning Journey

This repository contains my structured journey of learning Python, starting from basic concepts to building real-world projects.

---

## Overview

The goal of this repository is to develop a strong foundation in Python and gradually move towards writing clean, modular, and application-level code.

---

## Topics Covered

* Basics (variables, data types, input/output, operators)
* Control Flow (if-else, loops, conditions)
* Functions (parameters, return values, recursion, lambda)
* Data Structures (lists, tuples, sets, dictionaries)
* Object-Oriented Programming (classes, inheritance, polymorphism)
* Modules and Packages
* File Handling (text files, JSON)
* Exception Handling

---

## Projects

* CLI To-Do Application
* Advanced To-Do App with JSON storage
* Flask-based To-Do Web App
